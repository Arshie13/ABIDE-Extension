# test-extension README

ABIDE Extension for VSCode. The purpose of this extension is to streamline the task management process for developers by having their project management tool within their code editor, thereby reducing context switching.

## Features

Task list with minimal functionality within the editor's activity bar.
Webview panel for viewing tasks and their status.
auto -branch-checkout when updating task status to in progress.
auto-generate commit message based on task description (not from code changes).
